# IP Query

A CLI tool for gathering IP and domain name information.

## Disclaimer

**Still in early development!**

## Requirements

ipq requires Python 3.6.1 or greater.

## Installation

```bash
pip install git+https://github.com/Jonxslays/ipq.git
```
